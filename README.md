# Restaurante União — Ceilândia Sul

Integrantes da dupla
- Kledilson Leôncio Pereira da Silva
- Lucas Monteiro Oliveira

Descrição do caso escolhido
O Restaurante União é um negócio de família que funciona desde 2002 na Ceilândia Sul, em Brasília. Eles servem almoço todos os dias (segunda a sábado) no sistema de self-service e são muito conhecidos na região pelo churrasco na brasa e pela comida caseira.

Necessidade identificada
Mesmo sendo um restaurante muito antigo e com muitos clientes, o União não tinha um site próprio. Eles dependiam só das redes sociais. O problema é que no Instagram as informações ficam espalhadas, e o cliente não achava fácil o cardápio, o endereço exato, o horário de funcionamento e o contato para pedir marmita.

Público-alvo
Moradores da Ceilândia Sul e trabalhadores da região que precisam almoçar ou pedir uma marmita gostosa e rápida durante o dia.

Objetivo do site
O objetivo do site é criar a identidade do restaurante na internet e dar as informações que o cliente procura rápido. A ação principal que queremos do visitante é que ele clique no link para ligar ou chamar o restaurante no WhatsApp.

Processo de desenvolvimento
Primeiro, nós conversamos sobre o que o restaurante precisava mostrar para o cliente. Depois, criamos a estrutura do site em HTML dividindo as seções de história, cardápio e contato. Usamos o Bootstrap para ajudar a deixar o site bonito no celular e finalizamos o visual com o CSS. No fim, colocamos o JavaScript para fazer as animações e postamos o site na internet.

Principais decisões do projeto
- Estrutura do HTML:Usamos as tags certas e organizadas (como nav para menu, section para as partes do site e footer para o rodapé) em vez de encher o código de "divs" sem sentido.
- Visual: Escolhemos letras fáceis de ler, cores que combinam com restaurante e organizamos tudo com bastante espaço para não ficar uma leitura cansativa.
-Celular (Responsivo):O site se adapta sozinho. Ele fica bonito tanto na tela do computador quanto na tela de um celular.
-Acessibilidade: Colocamos textos explicativos em todas as imagens (usando a tag alt) para ajudar quem usa leitores de tela.

Funcionalidade em JavaScript
Criamos quatro coisas simples e úteis com o JavaScript:
1. O menu do topo (Navbar) muda de cor quando o usuário rola a página para baixo, facilitando a leitura.
2. Quando você clica em um link do menu, a tela desliza suavemente até a seção (História, Cardápio, etc) e fecha o menu do celular sozinho.
3. Os elementos do site aparecem deslizando de forma suave na tela conforme você vai descendo a página.
4. O ano no rodapé do site atualiza sozinho de forma automática.

Uso de Bootstrap
Usamos o Bootstrap 5 de um jeito leve para ajudar na estrutura:
- Usamos os Containers e as Linhas (Rows/Cols) para organizar os cards e os contatos na tela.
- Usamos o menu dinâmico (Navbar) que vira aquele "botão de três linhas" no celular automaticamente.

Testes realizados
- Testamos o site em telas de vários tamanhos usando o inspecionar do navegador.
- Clicamos em todos os links (Instagram, Telefone) para ver se funcionavam.
- Testamos navegar no site usando apenas o teclado (tecla TAB).
- Postamos no GitHub Pages e vimos se as imagens e o visual carregavam certos na internet.
